## [](#header-2)Abstract

The MD5-SHA checker program calculates and compares the selected hash code of the choosed file. This Java program uses the [MessageDigest](https://docs.oracle.com/javase/7/docs/api/java/security/MessageDigest.html) class, and generate MD5, SHA-1, SHA-256, SHA-384, SHA-512 hash codes. The [md5hash.jar](../md5hash.jar) code runs width jre1.8.0_131 or higher version of JVM.

### [](#header-3)How to check hash codes?
1.  Choose a file or type the path and file name.
2.  Copy or type the added hash code.
3.  Calculate and check the selected hash value.
